# AI视频帧搜索引擎

影视素材片段搜索，电影片段搜索，短视频素材查找，剪辑素材搜索引擎，AI视频帧搜索引擎，根据文字描述匹配画面、视频帧搜索、根据画面描述搜索视频。

## 工具介绍

该工具的搜索引擎拥有强大的机器智能，通过关键词就能够迅速搜索到你需要的素材，为你省下大量的时间。

比如输入：雪山之巅，就能找到大量雪山相关的视频素材。
